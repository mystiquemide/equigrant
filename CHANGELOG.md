## [0.1.0] - 2026-05-19

### Added

- Initial public EquiGrant MVP for AI-governed grant and bounty workflows on GenLayer.
- Next.js App Router frontend with responsive landing, bounty, submission, creator, dashboard, and leaderboard routes.
- GenLayer Python intelligent contract with bounty creation, submission, evaluation, appeal, resolve, edit, pause, resume, extend, and delete flows.
- Direct-mode contract test suite and GenVM lint validation.
- GitHub Actions CI for contract lint, direct tests, frontend lint, type-check, and production build.
- CodeQL security analysis workflow.
- Dependabot configuration for npm and GitHub Actions dependency updates.
- Public README with product positioning, screenshots, architecture, quick start, verification, and deployment guidance.
- Contributing guide, security policy, deployment guide, issue templates, and pull request template.
